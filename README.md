# Azumi Analytics — Data & AI Consultancy

A modern, interactive website for Azumi Analytics, a Nairobi-based data consultancy firm.

## Features
- Interactive hero slider
- Particle network background
- AI-powered chat assistant
- Smart contact form with suggestions
- Animated counters
- Fully responsive design

## Technologies
- React (via CDN)
- Babel Standalone
- CSS3
- Font Awesome
